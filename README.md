# Chris Robb – Senior AI Engineer Portfolio

An interactive, modern portfolio website showcasing AI engineering projects, career milestones, and technical achievements.

🌐 **Live Website:** [robb-chris.github.io/Portfolio](https://robb-chris.github.io/Portfolio/)

## ✨ Key Features
- **Dynamic Gravity Mesh (`gravity.js`):** Interactive 2D physics simulation with mouse-influenced gravity wells, twinkling ambient stars, and particle connection networks.
- **Adaptive Day/Night Theme System:** Seamless light/dark mode switching with persisted localStorage preferences and responsive system color-scheme detection.
- **Interactive Career Timeline:** Responsive timeline with automatic year transition markers, quantified business impact metrics, and technology badges.
- **Modern Design Architecture:** Built with custom CSS glassmorphism cards, Google Fonts (Outfit & Inter), and responsive layouts without bulky UI frameworks.

## 🛠️ Technology Stack
- **Core:** HTML5, Vanilla JavaScript (ES6+), Canvas API
- **Styling:** Vanilla CSS3 (Custom design system, CSS Variables, Responsive Grid/Flexbox)
- **Deployment:** GitHub Pages
