# Portfolio Project Rules

## Workflow & Verification
- **Skip Automated Verification for Minor Adjustments:** When performing small incremental edits, CSS styling changes, opacity/transparency tweaks, or minor content updates, do NOT launch background test servers or invoke browser subagents for automated verification. Apply code edits directly and inform the user so they can test immediately on their local setup, conserving time and tokens.
- **Reserved Verification:** Only invoke browser subagent verification when implementing major architectural layouts from scratch or when explicitly requested by the user.
